Java (Programming)

What is the file extension for a Java source code file?
a. .txt
b. .java
c. .exe
d. .doc
Correct answer: b. .java

Which keyword is used to start a Java program?
a. begin
b. start
c. main
d. run
Correct answer: c. main

Which symbol is used to end a line of code in Java?
a. .
b. ;
c. :
d. ,
Correct answer: b. ;

Which keyword is used to create a new object?
a. make
b. new
c. build
d. create
Correct answer: b. new

What is the default value of an integer in Java?
a. 1
b. 0
c. null
d. -1
Correct answer: b. 0

Which keyword is used to inherit another class?
a. extends
b. inherits
c. super
d. copy
Correct answer: a. extends

What does JVM stand for?
a. Java Virtual Machine
b. Java Version Manager
c. Java Value Model
d. Java Variable Method
Correct answer: a. Java Virtual Machine

Which package is always available in Java?
a. java.io
b. java.util
c. java.lang
d. java.math
Correct answer: c. java.lang

Which keyword makes a variable constant?
a. static
b. final
c. fixed
d. lock
Correct answer: b. final

Which company created Java?
a. Microsoft
b. Sun Microsystems
c. Apple
d. Google
Correct answer: b. Sun Microsystems
