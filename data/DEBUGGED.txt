1. Which OOP principle allows a class to inherit the members of another class?
A) Encapsulation
B) Polymorphism
C) Inheritance
D) Aggregation
Correct Answer: C) Inheritance

2. Which PHP function retrieves the number of rows returned by a SELECT query?
A) mysql_rows()
B) mysql_num_rows()
C) mysql_total_rows()
D) mysql_fetch_rows()
Correct Answer: B) mysql_num_rows()

3. Which keyword is commonly used to override a virtual method in C#?
A) override
B) overload
C) replace
D) virtualize
Correct Answer: A) override

4. Which MySQL function retrieves the value of a single cell from a query result?
A) mysql_fetch_array()
B) mysql_result()
